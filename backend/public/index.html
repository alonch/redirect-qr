How it works

  1. Connect to a Bluetooth thermal printer
  2. Click "Generate & Print QR Code" to create and print a QR code
  3. Give the printed QR code to an attendee
  4. When first scanned, the attendee will be directed to a registration form
  5. After the attendee registers their URL, future scans will redirect directly to their URL

Each QR code can only be registered once. If an attendee needs a new code, generate a new one here.
